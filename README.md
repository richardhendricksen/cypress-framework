# Frontend integration tests
The frontend integration tests Cypress for testing and mocking.

### Running
Run tests using ```yarn test:ci```.  

### Cypress
Open Cypress using ```yarn cy:open```.  

### Mocks
Mocks are defined in the `fixtures` folder.
